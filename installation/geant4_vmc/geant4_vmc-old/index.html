Installing Geant4 VMC - Older Versions 

<li><b>OpenGL visualization </b> <br>

It is recommended to build Geant4 X11 OpenGL visualization driver used in the VMC examples. It is handled in Geant4 CMake build via the following CMake option:
-DGEANT4_USE_OPENGL_X11=ON
