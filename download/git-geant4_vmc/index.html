Geant4 VMC

Development version (the whole repository):

git clone http://github.com/vmc-project/geant4_vmc.git

To switch to the ‘pro’ tagged version 6.8 (compatible with vmc 2.1, Geant4 11.4):

cd geant4_vmc 
git checkout v6-8

To switch to the ‘old’ tagged version 6.7.p1 (compatible with vmc 2.x, Geant4 11.3.):

cd geant4_vmc 
git checkout v6-7-p1

The list of new developments, bug fixes and the required versions of ROOT and Geant4 for each version can be found in the history file.
